================================================================================
                          DATABASE SCHEMA
================================================================================

USAGE_RECORDS TABLE:
- id (PK)
- model (TEXT) - Claude model name
- input_tokens (INTEGER)
- output_tokens (INTEGER)
- total_tokens (INTEGER)
- cost (REAL) - Calculated cost
- timestamp (DATETIME) - When API was called
- conversation_id (TEXT) - Claude conversation ID
- source (TEXT) - 'claude_ai' or 'anthropic_api'

PRICING TABLE:
- id (PK)
- model (TEXT, UNIQUE) - Model name
- input_price (REAL) - $ per 1M tokens
- output_price (REAL) - $ per 1M tokens
- last_updated (DATETIME)
- source (TEXT) - 'anthropic' or 'manual'

INDEXES:
- idx_timestamp (on usage_records.timestamp)
- idx_model (on usage_records.model)
- idx_source (on usage_records.source)
- idx_pricing_model (on pricing.model)

================================================================================
                           API ENDPOINTS
================================================================================

USAGE ENDPOINTS:
- POST /api/usage/track
  Request: {model, input_tokens, output_tokens, conversation_id}
  Response: {success, id, cost}

- GET /api/usage/summary?period=day|week|month
  Response: {period, request_count, total_input_tokens, total_output_tokens,
             total_tokens, total_cost}

- GET /api/usage/models
  Response: {models: [{model, request_count, input_tokens, output_tokens,
                      total_tokens, total_cost}]}

- GET /api/usage/history?limit=50&offset=0
  Response: {records: [{id, model, input_tokens, output_tokens, total_tokens,
                       cost, timestamp, conversation_id}], limit, offset}

PRICING ENDPOINTS:
- GET /api/pricing
  Response: {pricing: [{id, model, input_price, output_price, last_updated,
                       source}]}

- PUT /api/pricing/:model
  Request: {input_price, output_price}
  Response: {success, message}

HEALTH:
- GET /health
  Response: {status: "ok"}
