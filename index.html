🚀 Quick Start

Step 1: Install Backend

Open terminal and navigate to the backend directory:

cd backend npm install npm run dev

✅ Backend will run on http://localhost:3000

Step 2: Setup Frontend

Open a new terminal and install frontend:

cd frontend npm install npm run dev

✅ Dashboard will run on http://localhost:5173

Step 3: Install Browser Extension

Load the extension in Chrome:

  1. Open Chrome and go to chrome://extensions
  2. Enable "Developer mode" (top right)
  3. Click "Load unpacked"
  4. Select the extension folder

✅ Extension will appear in your toolbar

Step 4: Start Tracking

Go to claude.ai and use Claude normally. The extension will automatically track your usage!

View your stats in the dashboard at http://localhost:5173
